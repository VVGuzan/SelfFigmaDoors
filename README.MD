Пример вёрстки на основе шаблона figma.

Для сравнения с шаблоном справа вверху располагается надпись, при наведении на которую поверх странички появляется экспортированный в png шаблон. Данная надпись также является ссылкой на сам шаблон на figma.com.

Можно заметить расхождения в некоторых значках и расстояниях между элементов. Это связано с тем, что шапка и подвал страницы в шаблоне - просто изображения. А расстояния между одинаковыми элементами почти всегда различаются. Поэтому для единообразия в большинстве случаев отступы были унифицированы.
В общем, не самый удачный шаблон для вёрстки.
